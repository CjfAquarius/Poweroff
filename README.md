# 一个“恶作剧”程序
强烈建议 <mark>三思而后运行</mark> ！！！   
该程序的建议用途：<mark>研究弹窗等WinAPI、线程管理</mark>
## 该程序会做什么
会随机取一个随机数，介于100~1000秒之间，然后弹出异步弹窗，并在3秒后执行关机指令   
全程除弹窗外不显示其它内容
## 目标平台
Windows 10 64位系统
## 误操作怎么办
方案一  在103秒之内键入Win+R，输入 *taskkill /im Poweroff.exe /f* 并确认  
方案二  在103秒之内打开任务管理器，找到详细信息一栏，找到*Poweroff.exe*结束任务
## 帮助
你可以自行编译，但建议将可执行文件重命名为*Poweroff.exe*，以根据误操作说明化除危险
